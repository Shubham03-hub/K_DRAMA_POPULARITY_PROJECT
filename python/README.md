# 📺 K-Drama Popularity Analysis

## 📌 Problem
Analyze factors driving popularity of K-Dramas using data analysis and visualization.

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI

## 📊 Key Insights
- Certain genres dominate popularity trends
- Viewer ratings strongly correlate with engagement
- Release year impacts popularity
